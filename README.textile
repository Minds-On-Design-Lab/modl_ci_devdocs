h1. A Developer Documentation package for CodeIgniter 2+.

h2. Install

# Copy <code>third_party/modl_devdocs</code> to your application's <code>third_party</code> folder.
# Copy <code>controllers/devdocs.php</code> into your application's <code>controllers</code> folder.
# Write your docs, save the file to <code>docs/devdocs.textile</code> and visit 'yoursite.com/devdocs' to see the results.
